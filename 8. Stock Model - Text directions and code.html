Generate Stock model:

rails g model Stock ticker:string name:string last_price:decimal

Create the table by running the migration:

rake db:migrate

Add gem 'stock_quote' to your gemfile and run bundle install --without production

IMPORTANT:Â Please review the stock_quote gem documentation in Github as the methods change from time to time

Hop on rails console to get stock price for a stock:

StockQuote::Stock.quote("symbol")

StockQuote::Stock.quote("symbol").op

StockQuote::Stock.quote("symbol").l # (This has now changed to .last_price, but subject to change again, please check the documentation on github)
